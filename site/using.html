Using

I find it fascinating and informative to learn what hardware, software and tools people use, so I'm sharing the same info. These things change over time, so this is not a fixed list.

Computers

  • Macbook Pro Used for day job, super great hardware but locked-in software
  • Pinebook Pro Low cost laptop Linux, open and modify-able. ARM architecture
  • iMac 2010 Given a new lease on life by being reformatted with Linux Found someone to donate it to!
  • Latte Panda Single board computer, previously used for Birdhouse project, running Linux
  • Old Macbook Pro Running 32bit Mac programs like EyeTv, for encoding VHS to digital
  • Raspberry Pi 2b Web server running this site
  • Raspberry Pi Zero Previously used for an iteration of the Birdhouse project
  • Raspberry Pi 3 Used with tiny screen to provide midi via ORCA to Synth
  • Intel/PC custom build Ethereum miner (I've since learned that cryptocurrency as a whole is bad for the environment)

Audio

  • ae modular
  • Zoom U-44 Audio interface
  • Pocket Operators

Photography

  • Pentax 35mm Purchased by my dad in the 70's. Completely manual, and very intuitive to use.
  • Nikon Coolpix P900 Extreme zoom in a point-shoot that looks like a DSLR, great for birding
